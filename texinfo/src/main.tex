\section{{Figures}}
Figures are also supported. The floats are directly compatible with \texttt{info}
\texttt{latex}. 
\texttt{makeinfo} allows to compile to \texttt{{-}{-}pdf,\ {-}{-}html,\ {-}{-}latex}.
Figure (see \hyperref[anchor:fig_003acaption]{figure~\ref*{anchor:fig_003acaption}}) shows that images can be imported.
When the format is \texttt{info} it looks for a \texttt{*.txt} file and when the
output is pdf it looks for any compatible format such as \texttt{*.png,*.pdf}.
